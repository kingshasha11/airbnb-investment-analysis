# Analyse du marché Airbnb

Projet Data Analyst visant à identifier les zones les plus rentables pour un investissement locatif Airbnb.

## Objectif du projet

Analyser les données Airbnb afin de déterminer :

- le prix moyen par quartier
- le taux d'occupation estimé
- les quartiers les plus chers
- les quartiers les plus demandés
- le revenu annuel estimé

## Technologies utilisées

- Python
- Pandas
- SQLAlchemy
- MySQL
- Matplotlib
- Jupyter Notebook

## Dataset

Base de données MySQL: airbnb_analysis

## Exemple d'analyse

Le projet calcule :

- prix moyen par quartier
- taux d'occupation estimé
- revenu annuel estimé
